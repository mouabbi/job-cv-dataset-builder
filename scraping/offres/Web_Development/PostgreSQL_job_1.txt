Title: Sr. Java / PostgreSQL Software Engineer
        Company: Beacon Hill
        Location: Miamisburg, Montgomery County
        
        Description:


        We are seeking highly skilled and experienced Senior Java Software Engineers to join our dynamic, global, enterprise development team. The ideal candidate will play a key role in solution design, implementation, and backend development while ensuring seamless database development, microservices architecture, and cloud integration. The role involves working on highly scalable, high-transaction applications, implementing CI/CD pipelines, and optimizing performance for large-scale distributed systems. Additionally, the candidate will be responsible for driving best practices in software architecture, ensuring scalability, reliability, and security in all aspects of development. The Senior Java Software Engineer will also contribute to technical discussions, mentor junior engineers, and collaborate closely with cross-functional teams to deliver high-quality software solutions.

Required Skills:

    Must have at least 7 or more years experience as a full stack, Java software engineer using Java 11+
    Seasoned developer with heavy experience with Spring Boot and Spring framework
    Must have strong experience developing complex microservices and APIs utilizing REST, SOAP including new and existing integrations
    Advanced Database development within PostgreSQL databases
    Experience and strong knowledge of containerized application development using tools such as Docker, Kubernetes, Helm
    Experience with build and dependency scripting tools such as Maven, GitHub, JFrog Artifactory
    Understanding of SLA-driven development in high-transaction environments
    Proficiency in JDBC for database connectivity and management
    Strong problem-solving skills and ability to work in an agile environment
    Excellent communication and teamwork skills

Desired Skills:

    Experience developing using Java 21 is a plus
    Oracle Database development skills are a nice to have
    Experience with Elasticsearch (legacy systems)
    Experience working within existing JSP code to provide occasional support on front end development
